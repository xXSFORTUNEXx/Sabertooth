.:Keyboard Controls:.

Movement:
W - UP
A - LEFT
S - DOWN
D - RIGHT

Direction:
I - LOOK UP
J - LOOK LEFT
K - LOOK DOWN
L - LOOK RIGHT

SPACE BAR - FIRE WEAPON
R - RELOAD
F - MELEE WEAPON
O - PICKUP ITEMS
E - INTERACT
M - SHOW/HIDE IN-GAME MENU
C - SHOW/HIDE CHAT WINDOW
B - SHOW/HIDE DEBUG WINDOW

ENTER - FOCUS CHAT
TAB - HIDE ALL UI
F12 - SCREENSHOT
ESC - QUIT TO DESKTOP

.:Gamepad Controls:.
LEFT JOYSTICK - MOVEMENT
RIGHT JOYSTICK - DIRECTION

A - PICKUP ITEM
Y - INTERACT
B - MELEE ATTACK
X - RELOAD
RIGHT TRIGGER - FIRE WEAPON
START - SHOW/HIDE IN-GAME MENU
BACK - SHOW/HIDE CHAT WINDOW
